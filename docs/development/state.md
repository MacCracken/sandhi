**1.6.5** ÔÇö 2026-06-17. **Live-network download gate + the two bugs it caught.** Closes the 1.6.4 loose end with `programs/_download_probe.cyr` ÔÇö a CI gate proving a real **redirected binary download round-trips to disk** end-to-end (follows githubÔćĺcodeload, streams to a temp fd via `sandhi_http_download`, re-reads the file, asserts `on-disk bytes == reported bytes` at status 200; skip-cleanly offline; argv(1) overrides the target).
